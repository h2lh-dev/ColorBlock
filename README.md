# ColorBlock

*ColorBlock est un jeu de logique où le joueur doit faire des choix pour remplir un tableau en suivant certaines règles*

## 1 - Principe du jeux.

Le but du jeu est de reproduire un modèle de couleur en ne pouvant utiliser que des pinceaux qui permettent de peindre soit les lignes, soit des colonnes selon l'emplacement du pinceau.


## 2 - Comment jouer.

###I  <-- 1 ceci peint la ligne

###I  <-- 2 ceci peint la ligne

###I  <-- 3 ceci peint la ligne

I I I I <-- 4,5,6 ceci peint la colonne

*Les pinceaux sont représenter par les I*

Le jeu se présente comme cela, et en entrant les différents indices des pinceaux, nous peignons la ligne ou la colonne du tableau à la couleur du pinceau.
